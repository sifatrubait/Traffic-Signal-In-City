OPTION 1: WINDOWS (with Visual Studio or MinGW)
═══════════════════════════════════════════════════════════════════════

Prerequisites:
  • Visual Studio 2019+ OR MinGW
  • GLUT library (freeglut or original)
  • OpenGL development files

Installation (Windows - MinGW):
  1. Download freeglut from: http://freeglut.sourceforge.net/
  2. Extract and add to MinGW paths
  
Compilation:
  g++ -o traffic_signal.exe traffic_signal_optimized.cpp ^
      -lfreeglut -lopengl32 -lglu32 -lm

Run:
  traffic_signal.exe


═══════════════════════════════════════════════════════════════════════
OPTION 2: LINUX (Ubuntu/Debian)
═══════════════════════════════════════════════════════════════════════

Install GLUT:
  sudo apt-get update
  sudo apt-get install freeglut3 freeglut3-dev

Compilation:
  g++ -o traffic_signal traffic_signal_optimized.cpp \
      -lglut -lGL -lGLU -lm -std=c++11

Run:
  ./traffic_signal


═══════════════════════════════════════════════════════════════════════
OPTION 3: MACOS
═══════════════════════════════════════════════════════════════════════

Install GLUT (if not already present):
  brew install freeglut

Compilation:
  clang++ -o traffic_signal traffic_signal_optimized.cpp \
          -framework GLUT -framework OpenGL -lm -std=c++11

Run:
  ./traffic_signal

QUICK START
═══════════════════════════════════════════════════════════════════════

1. Copy traffic_signal_optimized.cpp to your project folder

2. Install GLUT (if needed):
   - Windows: Download freeglut, add to PATH

3. Compile:

4. Run:
   ./traffic_signal  (or traffic_signal.exe on Windows)

5. Press 'f' to start
   Press 'r' or 'g' to toggle traffic signals
